# Day-One IT Setup

## Objective
Guide the first-day support session to verify the employee can work successfully.

## Day-One Tasks
1. Confirm the employee has received their assigned hardware.
2. Verify sign-in to workstation using company credentials.
3. Enroll or validate MFA.
4. Confirm email access on desktop and web.
5. Test collaboration tools.
6. Validate VPN connection if required.
7. Confirm required business applications launch.
8. Verify printer access if relevant.
9. Explain password reset and IT support contact methods.
10. Confirm user understands basic security expectations.

## Success Criteria
- User is able to sign in
- User can access email and required tools
- User can reach internal resources if needed
- No blockers remain for normal job duties

## Escalate When
- Core account is not provisioned
- Required license is missing
- Business-critical system access is not available
- Device enrollment or security controls fail
