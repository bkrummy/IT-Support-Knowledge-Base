# Laptop Handoff Procedure

## Objective
Standardize the preparation, assignment, and acknowledgment of end-user hardware.

## Procedure
1. Verify device record in inventory
2. Confirm device is wiped or newly provisioned
3. Apply asset tag if required
4. Enroll device in endpoint management
5. Install standard software set
6. Verify encryption and security baseline
7. Confirm assigned user details
8. Record serial number and asset tag in handoff form
9. Obtain acknowledgment from employee
10. Update inventory status to assigned

## Required Records
- asset tag
- serial number
- assigned employee
- handoff date
- condition notes
- included accessories
