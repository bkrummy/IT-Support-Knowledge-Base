# Manager Preboarding Checklist

## Objective
Collect the information IT needs to prepare a new hire accurately and on time.

## Required Information from Hiring Manager
- Employee full name
- Start date
- Job title
- Department and cost center
- Work location or remote status
- Manager name
- Required applications
- Required shared mailbox, shared drive, or distribution list access
- Required hardware or accessibility needs
- Whether elevated or privileged access is needed

## Manager Actions
- Submit onboarding request within agreed lead time
- Provide all required access details
- Confirm whether user needs VPN access
- Confirm if user requires mobile device or phone setup
- Confirm seating location if onsite
- Identify any time-sensitive access needed for day one

## IT Review Criteria
- Request is complete
- Requested access is role-appropriate
- Approvals are included where needed
- Hardware request is standard or properly justified
- Exceptions are documented

## Common Delays
- Missing manager approval
- Missing start date or location
- Unclear application list
- Privileged access requested without business justification
