# Account Disablement Checklist

## Objective
Ensure access is removed in the correct order and at the correct time.

## Systems to Review
- Directory account
- Email account
- VPN account
- Collaboration tools
- Shared mailbox permissions
- Business applications
- MFA registration
- Admin or privileged access

## Checklist
- Confirm disablement timing
- Disable primary identity account
- Revoke active sessions if required by policy
- Disable VPN and remote access
- Remove privileged group membership
- Transfer mailbox or delegate access as approved
- Remove access to shared resources
- Document all completed steps

## Notes
For involuntary terminations, follow company security and HR guidance for exact timing and notification handling.
