# Wired Network Troubleshooting

## Symptoms
- No network connectivity on dock or Ethernet
- Intermittent connection
- Link light not present
- Works on Wi-Fi but not wired

## Troubleshooting Steps
1. Confirm cable is connected firmly at both ends.
2. Test alternate port or cable if available.
3. Check dock status and power if using a dock.
4. Confirm network adapter is enabled.
5. Renew DHCP lease or restart adapter.
6. Test another known-good device on same port if appropriate.
7. Escalate if switch port or building network issue is suspected.
