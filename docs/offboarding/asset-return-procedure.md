# Asset Return Procedure

## Objective
Standardize recovery of company-owned hardware and accessories during offboarding.

## Assets Commonly Recovered
- Laptop
- Docking station
- Monitors
- Keyboard and mouse
- Headset
- Mobile phone
- Access badge
- Charger and adapters

## Procedure
1. Review assigned assets in inventory system.
2. Coordinate return date and method.
3. Inspect returned equipment for visible damage.
4. Confirm serial numbers and asset tags.
5. Record missing items or condition concerns.
6. Update inventory to returned or pending evaluation.
7. Route device for wipe, reprovisioning, or disposal.

## Exceptions
- Remote employee shipping delays
- Lost or damaged equipment
- Shared devices needing manager review
