# Knowledge Base Review Cycle

## Objective
Ensure documentation remains accurate and useful over time.

## Recommended Review Cadence
- Quarterly review for high-use support articles
- Semiannual review for lower-use procedures
- Immediate review after major system or process changes

## Review Checklist
- Steps still accurate
- Screens or workflows still match reality
- Escalation path still valid
- Ownership still clear
- Broken links removed
- Duplicates consolidated
