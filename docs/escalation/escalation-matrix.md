# Escalation Matrix

## Purpose
Define when frontline support should resolve, escalate, or coordinate handoff to another team.

## Level 1 Support Handles
- Password resets
- Basic email support
- Basic VPN troubleshooting
- Printer setup and common failures
- Standard software installs
- New hire setup coordination
- Asset handoff tasks

## Escalate to Level 2 or Infrastructure When
- Repeated issue persists after documented troubleshooting
- Multiple users or business units are affected
- Server, network core, identity platform, or tenant-level fault is suspected
- Administrative access is required beyond support scope
- Security concern is identified

## Escalate to Security When
- Phishing or suspected compromise is reported
- Unauthorized access is suspected
- Sensitive data exposure may have occurred

## Escalation Standards
- Include summary of issue
- Include affected user(s)
- Include business impact
- Include troubleshooting already performed
- Include screenshots or logs where appropriate
- Include urgency and timing constraints
