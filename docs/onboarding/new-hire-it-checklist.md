# New Hire IT Checklist

## Objective
Ensure new employees receive the correct devices, access, and baseline support before or on their first day.

## Inputs Required
- Employee full name
- Start date
- Department
- Manager name
- Job title
- Work location
- Required systems and access
- Hardware requirements

## Checklist
- Confirm approved hire request from HR or hiring manager
- Verify start date and location
- Create or confirm user account
- Assign email license and collaboration tools
- Prepare laptop and required peripherals
- Apply standard security controls and endpoint management
- Add user to required distribution lists and groups
- Submit or complete application access requests
- Confirm MFA enrollment steps are ready
- Prepare welcome instructions
- Schedule day-one support session if needed

## Day-One Validation
- User can sign in successfully
- Email is working
- MFA is enrolled
- VPN is configured if required
- Required business applications launch successfully
- Printer access is available if needed
- User knows how to contact IT support

## Completion Criteria
All required hardware, accounts, and baseline access are delivered and tested.
