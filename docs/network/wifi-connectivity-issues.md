# Wi-Fi Connectivity Issues

## Symptoms
- Cannot join wireless network
- Connected with no internet
- Frequent drops
- Authentication failure

## Basic Checks
- Confirm correct SSID
- Confirm password or certificate-based auth requirements
- Forget and reconnect to network
- Test another device on same network
- Check whether wired connection works

## Escalate When
- Multiple users are affected in same area
- Access point outage is suspected
- Corporate certificate or NAC issue is suspected
