# Workstation Slow Performance

## Common Symptoms
- Long startup time
- Applications freeze or crash
- High CPU or memory usage
- Slow login
- Browser or collaboration tools lag

## Basic Troubleshooting
1. Ask when the slowdown started.
2. Determine whether the issue affects one app or the entire device.
3. Restart the machine if uptime is excessive.
4. Check available disk space.
5. Review Task Manager or Activity Monitor for high resource use.
6. Confirm endpoint security scan is not causing temporary load.
7. Remove unnecessary startup items if permitted.
8. Install pending updates if approved and safe.
9. Escalate if hardware health or broader endpoint issue is suspected.

## Escalate When
- Repeated crashes occur
- Disk health warnings appear
- Performance degradation affects multiple machines of same type
- Hardware replacement may be needed
