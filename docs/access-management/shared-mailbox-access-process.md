# Shared Mailbox Access Process

## Objective
Standardize shared mailbox access requests and reduce permission errors.

## Required Information
- Mailbox name
- User requiring access
- Access type needed
- Manager or mailbox owner approval
- Duration if temporary

## Process
1. Confirm mailbox exists and correct name is provided.
2. Validate approval.
3. Apply appropriate permission level.
4. Inform user that auto-mapping may take time if applicable.
5. Document completion in ticket.

## Common Issues
- Wrong mailbox name provided
- Approval missing
- User expects mailbox immediately before sync completes
