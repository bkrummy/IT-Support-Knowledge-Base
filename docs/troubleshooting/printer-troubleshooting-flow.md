# Printer Troubleshooting Flow

## Common Symptoms
- Printer appears offline
- Print jobs stuck in queue
- Wrong printer selected
- Driver errors
- User can print from one app but not another

## Troubleshooting Steps
1. Confirm printer name and location
2. Check whether issue affects one user or many
3. Verify printer has power and network connection
4. Confirm device is connected to the correct network
5. Clear stuck print jobs
6. Restart print spooler if needed
7. Re-add printer or reinstall driver if appropriate
8. Test print from a simple application
9. Escalate for hardware fault or printer server issue

## Escalate When
- Multiple users cannot print to same printer
- Printer has recurring hardware errors
- Queue issue persists after spooler reset
- Print server or permissions issue is suspected
